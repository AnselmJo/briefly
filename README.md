# Briefly

Persönliches tägliches Audio-Briefing ("Daily Cast"): eine automatisch
generierte, ~10-minütige Sprachfolge aus News, Wissens-Themen und eigenen
Inbox-Einträgen, ausgeliefert als abonnierbarer RSS-Feed.

> „Briefly – deine Themen, kurz und persönlich gesprochen, jeden Morgen."

Aktueller Stand: Projekt-Grundgerüst (Phase 0). Die eigentliche
Pipeline-Implementierung (Ollama, TTS, RSS-Sammlung, Feed-Generierung) folgt
in Phase 1 – siehe [`CLAUDE.md`](./CLAUDE.md) und
[`docs/concept/01-briefing-fuer-claude-fable.md`](./docs/concept/01-briefing-fuer-claude-fable.md).

Setup-Anleitung folgt, sobald die Pipeline lauffähig ist (Definition of Done,
Briefing §11).
